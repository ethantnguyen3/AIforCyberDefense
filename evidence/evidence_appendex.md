# Evidence Appendix

This appendix summarizes the evidence files used to support the final AIforCyberDefense project.

| Evidence File | Purpose |
|---|---|
| before_after_comparison.md | Shows baseline versus after-controls comparison |
| validation_test_table.md | Shows validation tests and outcomes |
| guardrail_test_results.md | Shows prompt guardrail testing |
| retrieval_hardening_results.md | Shows retrieval hardening results |
| least_privilege_matrix.md | Shows access control and role restrictions |
| logging_traceability_design.md | Shows logging and audit trail design |
| eval/baseline_results.csv | Shows baseline testing results |
| eval/controlled_results.csv | Shows after-control testing results |
| tests/test_controls.py | Shows unit tests for defensive controls |
